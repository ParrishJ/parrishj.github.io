Why I Made This Demo

Pursuing web and software development as a profession requires a strong set of skills that span many different domains. As part of my ongoing effort to hone and demonstrate my skills in some of these different areas, I’m publishing different projects that focus on particular technologies.

This project is focused on Bootstrap 4. Aesthetics are important to any project on the web, and Bootstrap offers a powerful and developer-friendly way to develop user facing content. Bootstrap’s immense popularity and developer-friendly design inspired me to put this project together.

Reasons to Love Bootstrap

Popular, Reliable, and Developer Friendly
• Immensely popular with companies of all sizes
• Mobile first design allows encourages developers to create elegant, clean interfaces that work on all devices
• Bootstrap's use of flexbox allows developers to create layouts intuitiely
• Bootstrap classes allow for developers to create stunning and professional designs without writing excessive amounts of css
